## 🎯 立即使用

### 1. 重启应用
```bash
# 停止旧进程
pkill -f "python.*app.py"

# 启动应用
cd /Users/apple/Downloads/wechat-mp-tools
python3 app.py --port 5200
```

### 2. 访问界面
打开浏览器：**http://localhost:5200**

### 3. 测试新功能

#### ✨ 推荐视频
- 点击"推荐视频"卡片
- 浏览推荐流
- 点击"下载视频"按钮下载

#### ✨ 收藏视频
- 点击"收藏视频"卡片
- 查看收藏列表
- 一键下载

#### ✨ 点赞视频
- 点击左侧菜单（如果有）或仪表盘
- 查看点赞列表
- 一键下载

#### ✨ 取消下载
- 进入"解析与下载"
- 选择"用户主页批量下载"
- 开始下载后，点击"取消下载"按钮
